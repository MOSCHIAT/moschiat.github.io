# Gerador de Citações

Um projeto Python simples que gera citações aleatórias inspiradoras.

## 🚀 Como usar

### Instalação

```bash
pip install -r requirements.txt
```

### Executar

```bash
python quotes_generator.py
```

## 📋 Estrutura do Projeto

```
python-project/
├── quotes_generator.py    # Script principal
├── requirements.txt       # Dependências
└── README.md             # Este arquivo
```

## 🎯 Funcionalidades

- Gera citações aleatórias
- Interface simples e limpa
- Fácil de estender
